## getBlockCommitment RPC Method

Returns commitment for a particular block in the ledger.

<div className="grid grid-cols-2 gap-1 mt-2">
  <div className="col-span-1">
    #### Parameters

    `u64` **required**\
    Slot number, as u64 integer

    `object` **optional**\
    Configuration object (currently no additional configuration options)
  </div>

  <div className="col-span-1">
    #### Code Sample

    ```bash
    curl https://rpc.testnet.soo.network/rpc \
    -X POST \
    -H "Content-Type: application/json" \
    -d '{
      "jsonrpc": "2.0",
      "id": 1,
      "method": "getBlockCommitment",
      "params": [369624]
    }'
    ```

    #### Response

    ```json
    {
      "jsonrpc": "2.0",
      "result": {
        "commitment": [
          100000000,
          200000000,
          300000000,
          400000000
        ],
        "totalStake": 500000000
      },
      "id": 1
    }
    ```
  </div>
</div>

#### Result Details

The result field will be:

* `<null>` - if specified block is not found or unknown

* `<object>` - if block is found, containing:

  * `commitment: <array>` - array of u64 integers representing:

    * The cumulative stake (in lamports) that has voted on the block

    * Depths range from 0 to `MAX_LOCKOUT_HISTORY` + 1

    * Each subsequent element represents additional stake confirmation

  * `totalStake: <u64>` - Total active stake in lamports for the current epoch

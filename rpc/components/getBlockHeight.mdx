## getBlockHeight RPC Method

Returns the current block height of the node.

<div className="grid grid-cols-2 gap-1 mt-2">
  <div className="col-span-1">
    #### Parameters

    `object` **optional**\
    Configuration object containing the following fields:

    **commitment** `string` **optional**\
    The level of commitment desired when querying state

    **minContextSlot** `number` **optional**\
    The minimum slot that the request can be evaluated at
  </div>

  <div className="col-span-1">
    #### Code Sample

    ```bash
    curl https://rpc.testnet.soo.network/rpc -X POST -H "Content-Type: application/json" -d '{
      "jsonrpc": "2.0",
      "id": 1,
      "method": "getBlockHeight",
      "params": [{
        "commitment": "finalized",
        "minContextSlot": 0
      }]
    }'
    ```

    #### Response

    ```json
    {
      "jsonrpc": "2.0",
      "result": 369624,
      "id": 1
    }
    ```
  </div>
</div>

#### Result Details

The result will be:

* `<u64>` - Current block height of the node

  * Represents the number of blocks beneath the highest block in the blockchain

  * Does not include the genesis block
